# gchure.bio

Repo with for my personal website, built with [Astro](https://astro.build).

## Structure

```
src/
├── components/     # Reusable Astro components
├── content/        # Markdown content (publications, writing, research, etc.)
├── layouts/        # Page layouts
├── pages/          # File-based routing
├── styles/         # Global CSS
└── utils/          # Shared utilities
public/
├── images/         # Static images
└── pdfs/           # Publication PDFs
```

## Development

```bash
npm install
npm run dev
```
